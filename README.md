# 프린세스 커넥트 리다이브 공략글 스크랩핑 프로젝트

## 디렉토리 구성
### /frontend
#### /view
* 설명 :프리코네 리다이브 공략을 확인 할 수 있는 웹사이트
* Language, Framework : vue.js

### /backend
#### /scraping
* 설명 : 디시인사이드 프리코네 리다이브 갤러리의 공략글을 scraping 하는 모듈
* Language : Phython

#### /api
* 설명 :scraping한 정보를 확인 할 수 있는 웹사이트 api 서버
* Language, Framework : java, spring
